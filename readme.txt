= Best Youtube Video Lazyload =

Youtube Video Lazyload improves Google page Speed Insights Score, Gtmetrix, and Pingdom score. Also, fix <strong>Reduce the impact of third-party Code</strong> & <strong>Some third-party resources can be lazy-loaded with a facade<strong> in page speed insights.

This time Youtube Video LazyLoad provides only a shortcode. You can use it anywhere for Youtube videos. You can pass the video id & custom thumbnail.

= Features of Best Youtube Video Lazyload =

* Quick to set up
* Set a custom preview image per video
* Shortcode to show anywhere
* Increase Google page speed, GTmetrix & Pingdom Tools Score.
* Reduce the impact of third-party code
* Reduce External requests.

== Shortcode ==
[youtube_lazyload id="youtube_video_id_here" thumb="custom_thumbnail_link"]
id attr required.
thumb attr optional.

== Installation ==

1. Install Best Youtube Video Lazyload by uploading the `best-youtube-video-lazyload` directory to the `/wp-content/plugins/` directory. 
2. Activate Best Youtube Video Lazyload through the `Plugins` menu in WordPress.
